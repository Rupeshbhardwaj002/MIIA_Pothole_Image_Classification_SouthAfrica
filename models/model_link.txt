model is hosted on hugging face you can download it from the link 
link -> https://huggingface.co/rupesh002/pothole_detection_model

To load model run this command in collab or notebook -> 
""
from huggingface_hub import hf_hub_download
import joblib

model_path = hf_hub_download(
    repo_id="rupesh002/pothole_detection_model",
    filename="model.pkl",
    repo_type="model"
)

model = joblib.load(model_path)
  ""
